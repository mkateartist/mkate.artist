MKate Gallery — статический сайт (портфолио + магазин)

1) Открой index.html локально (двойной клик) или залей папку на Netlify.
2) Контент работ лежит в:
   - works.json (универсальный JSON)
   - data.js (подключается на сайт)

Как заменить/добавить работы:
- Добавь изображения в assets/works/
- Отредактируй works.json (или data.js), затем обнови data.js из works.json (по желанию).
